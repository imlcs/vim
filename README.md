# vim
vim简易配置文件

```
ctrl + w : 进入和退出 paste 模式(粘贴代码段的时候用,否则会出现自动缩进的情况)
ctrl + s : 保存文件
ctrl + a : 复制文本全部内容, 按 p 键粘贴
F12      : 代码格式化
lh       : 在行首进行插入
le       : 在行尾进行插入
s        : 搜索单词词首 2 个字母, 然后按显示的红色字母进行跳转
,j       : 向下跳转行
,k       : 向上跳转行
,w       : 向行后跳转到单词词首
,b       : 向行前跳转到单词词尾
,cs      : 块注释
,cu      : 取消块注释
```
